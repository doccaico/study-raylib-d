## lifegame

### Setting
```sh
$ dub upgrade
$ dub raylib-d:install
```

### Debug
```sh
# build
$ dub build

# build and run
$ dub
```

### Release
```sh
# build
$ dub build -b release-nobounds

# build and run
$ dub -b release-nobounds
```
